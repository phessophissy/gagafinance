# Gagafinance Frontend

This is the frontend for the Gagafinance DeFi application, built with Vite, React, TypeScript, and Stacks.js.

## Tech Stack

- **Framework:** Vite + React + TypeScript
- **Styling:** Tailwind CSS
- **Blockchain Interaction:** @stacks/connect, @stacks/network
- **Icons:** Lucide React
- **Animations:** Framer Motion

## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```

3.  **Build for Production:**
    ```bash
    npm run build
    ```

## Project Structure

- `src/components`: Reusable UI components.
- `src/pages`: Route components.
- `src/hooks`: Custom React hooks (e.g., wallet connection, contract interaction).
- `src/utils`: Utility functions.
